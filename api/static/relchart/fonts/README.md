This folder is for self-hosted fonts used by the relchart v3 frontend.

To use the same Garamond styling as your other site, copy these files from:
https://github.com/TaliesinDS/TaliesinDS.github.io/tree/main/assets/fonts

Recommended minimum set:
- EBGaramond-Regular.woff2
- EBGaramond-Italic.woff2
- EBGaramond-Bold.woff2
- EBGaramond-BoldItalic.woff2

Optional (if you want weight 500/600/etc):
- EBGaramond-Medium.woff2
- EBGaramond-MediumItalic.woff2
- EBGaramond-SemiBold.woff2
- EBGaramond-SemiBoldItalic.woff2
- EBGaramond-ExtraBold.woff2
- EBGaramond-ExtraBoldItalic.woff2

After copying, refresh the page; the panel will use the embedded font automatically.
